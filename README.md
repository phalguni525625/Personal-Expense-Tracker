# Personal-Expense-Tracker
A Java-based Personal Expense Tracker that helps users record, manage, and analyze daily expenses. The application supports expense categorization, monthly expense reporting, highest spending category analysis, and persistent data storage using file handling and serialization.
# Smart Password Generator

A Java application that generates secure passwords from user information and evaluates password strength.

## Features
- Accepts Name, PAN Number, and Date of Birth
- Generates unique secure passwords
- Classifies passwords as Weak, Medium, or Strong
- Supports multiple users
- Console-based interface

## Technologies Used
- Java
- OOP Concepts
- String Manipulation
- Random Number Generation
